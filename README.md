# 🐳 AI Dockerfile Generator using Google Gemini
A GenAI-powered tool that generates optimized Dockerfiles based on the programming language provided by the user. This project uses the Google Gemini API to generate production-ready Dockerfiles following Docker best practices.
## 📋 Prerequisites

### Install Python

Download Python (3.10 or later)

https://www.python.org/downloads/

Verify installation

```bash
python --version
```

or

```bash
python3 --version
```

---
## 🚀 Project Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ai-dockerfile-generator.git
cd ai-dockerfile-generator
```

### 2. Create Virtual Environment

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Run the Application

```bash
python generate_dockerfile.py
```
